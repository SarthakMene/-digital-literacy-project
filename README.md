📌 Project Overview

This project is part of the Digital Literacy course at VIT Bhopal University. As a Student Digital Ambassador, the objective is to create a digital portfolio that helps students understand essential digital skills such as online presence, cybersecurity, communication, and collaboration tools.

The repository contains all five tasks covering different modules of digital literacy along with a detailed project report.



📊 Task Summary
🧩 Task 1 – Digital Literacy Infographic

Created an infographic explaining digital literacy, safe internet practices, and useful tools using Canva.

🌐 Task 2 – Student Digital Portfolio

Set up professional profiles on platforms like GitHub, LinkedIn, and others to build an online presence.

💻 Task 3 – Coding & Collaboration Platforms
Completed a beginner coding challenge on HackerRank/CodeChef
Created a Digital Literacy Awareness Quiz using Google Forms
